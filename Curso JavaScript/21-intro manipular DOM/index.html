Manipulando o DOM

Nessa aula realizamos o acesso à uma pagina web e inspecionamos o campo de comentários
Depois no console inserimos no console o comando:
document.getElementById('comment').innerText = 'Texto inserido';
Para inserir um texto no campo comentário.
E document.getElementById('submit').click() para publicá-lo
É possível também editar um elemento do site com
document.getElementsByClassName('nome da classe do elemento')[0].innerText = 'insere texto'
Obs: entre [numero do indice].
