Factory functions

Geralmente as funções de "fabrica" são utilizadas para encapsular
a informação dentro de um metodo que facilita a criação de diversos objetos
do mesmo tipo
Exemplo:
Ao invés de copiar diversas vezes uma função para "criar um celular"
você encapsula o metodo "criarCelular" para facilitar a criação dos futuros "celulares"
